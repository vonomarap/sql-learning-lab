# Contributing

## File Naming

- Prefix ordered examples with two digits: `01_`, `02_`, `03_`
- Use lowercase kebab or snake case after the prefix
- Keep one topic per file

## Directory Rules

- `scripts/` contains runnable examples
- `notes/` contains concise explanations and mental models
- `exercises/` contains a `task.md` and `solution.sql`
- `snippets/` contains copy-paste patterns
- `datasets/` contains training data and restore instructions

## Style

- Prefer clear aliases
- Keep examples short and focused
- Add comments only when a query step is not obvious
- Avoid mixing multiple topics in one file
