# Oracle SQL Homework Archive

This directory stores a historical Oracle SQL homework bundle as a single archive block.

## Included Materials

- SQL topic files and homework solutions
- homework PDFs
- slide deck PDF
- `plan.doc`

## Notes

- The files are preserved verbatim from the source archive
- Original file names are intentionally kept unchanged
- The archive mixes reference material and homework assets in one place by design

## Purpose

This bundle documents earlier Oracle study materials without forcing them into the curated PostgreSQL and generic SQL structure used elsewhere in the repository.
