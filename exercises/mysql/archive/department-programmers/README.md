# Department Programmers Exercise Archive

This directory stores historical text-based SQL materials related to the `department_programmers` dataset.

## Included Files

- `exam-sql-tasks.txt`
- `test-sql-queries.txt`

## Notes

- The files are preserved verbatim from the source archive
- They are treated as exercise and query packs, not curated notes
- Original wording and encoding are intentionally kept as-is

## Suggested Usage

1. Load one of the `department_programmers` SQL dumps from `datasets/mysql/department-programmers/`
2. Open these text files as reference tasks or query collections
3. Rewrite or solve selected tasks later as curated `.sql` or `.md` materials
